# MVP版本使用指南

## 🎉 欢迎使用财富管理智能体 MVP版本

本版本已完成核心功能的开发，可以立即体验！

## 🚀 快速开始

### 1. 启动应用

```bash
cd wealth-agent
npm run dev
```

### 2. 注册账号

首次使用需要注册账号：
1. 点击"注册"标签
2. 输入邮箱地址
3. 输入密码（至少6位）
4. 点击"注册"按钮

### 3. 体验核心功能

注册登录后，你可以体验以下功能：

#### 📊 资产总览
- 查看净资产、总资产、总负债统计
- 查看资产分布饼图
- 查看资产构成柱状图

#### 💰 资产管理
- 点击"资产管理"Tab
- 点击"添加资产"按钮
- 选择资产类型（现金/股票/基金/房产/负债）
- 填写资产信息
- 点击"添加"保存

#### 📈 持仓管理
- 点击"持仓管理"Tab
- 点击"添加持仓"按钮
- 选择持仓类型（股票/基金）
- 输入股票代码、名称、数量、成本
- 点击"添加"保存

#### 🤖 AI投顾
- 点击"AI投顾"Tab
- 输入你的问题
- 按Enter发送
- 查看AI的回复

**推荐问题：**
- "我的投资风险如何？"
- "如何优化我的资产配置？"
- "我的持仓表现怎么样？"
- "总资产是多少？"

## 🔑 功能说明

### 资产类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 现金 | 银行存款、现金 | 活期存款、定期存款 |
| 股票 | A股、港股、美股 | 贵州茅台、腾讯控股 |
| 基金 | 公募基金、私募基金 | 易方达消费行业 |
| 房产 | 住宅、商铺 | 自住房、投资房 |
| 负债 | 房贷、车贷、信用卡 | 房屋贷款、信用卡欠款 |

### 货币支持
- 人民币 (CNY) - 默认
- 美元 (USD)
- 欧元 (EUR)
- 港币 (HKD)
- 日元 (JPY)

### 盈亏计算
- 盈亏金额 = (当前价 - 成本价) × 数量
- 盈亏比例 = 盈亏金额 / 成本金额 × 100%
- 盈利显示绿色，亏损显示红色

## ⚙️ AI功能说明

### 无API Key模式
如果没有配置OpenAI API Key，应用会使用内置的智能回复：
- 支持风险评估分析
- 支持资产配置建议
- 支持持仓表现解读
- 支持一般性投资问答

### 有API Key模式
配置OpenAI API Key后，可以获得更智能的对话体验：
1. 创建 `.env` 文件
2. 添加 `VITE_OPENAI_API_KEY=your-api-key`
3. 重启开发服务器

## 📱 响应式设计

应用支持不同屏幕尺寸：
- 桌面端（>1200px）：完整布局
- 平板端（768-1200px）：自适应布局
- 移动端（<768px）：垂直堆叠布局

## 🔒 数据安全

- 用户密码使用bcrypt加密
- 数据存储在浏览器本地（localStorage）
- 每个用户的数据完全隔离
- 刷新页面后数据保持

## 🐛 常见问题

### Q: 数据保存在哪里？
A: 数据保存在浏览器的localStorage中，刷新或关闭浏览器不会丢失。

### Q: 如何清除所有数据？
A: 在浏览器开发者工具中清除该站点的localStorage。

### Q: AI回答不准确怎么办？
A: MVP版本使用简化模型，正式版会接入真实API提升准确性。

### Q: 支持哪些浏览器？
A: 支持Chrome、Firefox、Edge、Safari等现代浏览器。
