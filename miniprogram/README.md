# Wealth Terminal - 微信小程序（WebView 版）

## 项目概述

这是 Wealth Terminal 的微信小程序版本，采用 **WebView 容器方案**：
- 小程序本身只是一个空壳，只有一个 `<web-view>` 组件
- 加载现有的 H5 网页：`https://wealth-agent.pages.dev`
- 所有功能都在 H5 端实现，小程序无需频繁更新

**AppID**：`wx4a767f9e2991590c`

## 快速开始

### 1. 准备工作

| 事项 | 说明 |
|------|------|
| 微信小程序账号 | 已注册，AppID: wx4a767f9e2991590c |
| 微信开发者工具 | [下载地址](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) |

### 2. 导入项目

1. 打开「微信开发者工具」
2. 选择「小程序」→「导入项目」
3. 项目目录选择：`miniprogram/` 文件夹
4. AppID 已预填在 `project.config.json` 中
5. 项目名称：Wealth Terminal
6. 点击「确定」

### 3. 配置业务域名（重要！）

WebView 加载的域名必须在小程序后台配置白名单，否则无法加载。

**操作步骤：**

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」→「开发管理」→「开发设置」
3. 找到「业务域名」，点击「修改」
4. 添加你的 H5 域名（如 `wealth-agent.pages.dev`）
5. 下载校验文件（文件名类似 `MP_verify_xxxxx.txt`）
6. 把校验文件放到 H5 网站的根目录（`public/` 文件夹）
7. 部署 H5 后，回到小程序后台点击「保存」

> 注意：域名必须是 HTTPS，且需要 ICP 备案

### 4. 修改 H5 地址

如果你的 H5 域名不是 `wealth-agent.pages.dev`，需要修改：

**文件：** `miniprogram/app.js`

```javascript
globalData: {
  webViewUrl: 'https://你的域名.com'  // 改成你的地址
}
```

## 提交审核上线

### 1. 上传代码

1. 在微信开发者工具中，点击右上角「上传」
2. 填写版本号（如 1.0.0）和项目备注
3. 点击「上传」

### 2. 提交审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「版本管理」
3. 找到刚上传的版本，点击「提交审核」
4. 填写审核信息：
   - **功能页面**：选择首页
   - **功能描述**：个人财富管理工具，支持资产记账、持仓管理、数据分析
   - **测试账号**：如果需要登录，提供测试账号
5. 提交审核

### 3. 类目选择建议

| 类目 | 说明 |
|------|------|
| 工具 → 记账 | 推荐：个人记账工具，符合个人主体 |
| 工具 → 信息查询 | 备选：如果记账类目不通过 |

**注意事项：**
- 小程序名称建议：财富助手、记账助手、财富管理工具 等
- 简介突出「个人记账/资产记录」，弱化金融属性
- 避免出现「股票分析」「AI 荐股」「投资建议」等敏感词

### 4. 发布上线

审核通过后：
1. 进入「版本管理」
2. 找到审核通过的版本
3. 点击「发布」
4. 确认发布，小程序正式上线

## 常见问题

### Q: 小程序上架后，H5 更新需要重新审核吗？

不需要。小程序只是一个 WebView 容器，H5 端的所有更新都会自动同步，用户下次打开小程序就能看到最新版本。

### Q: 可以分享到朋友圈吗？

WebView 小程序支持分享给好友和群聊，但分享到朋友圈的功能受限。建议通过好友/群聊传播。

### Q: 可以用微信登录吗？

WebView 内的网页可以走微信网页授权（OAuth），但比原生小程序登录复杂一些。当前版本使用邮箱登录，不受影响。

### Q: 可以用微信支付吗？

WebView 内可以用 H5 支付，但需要额外对接。如果是放收款码让用户转账，则不需要微信支付。

### Q: 加载慢怎么办？

首次打开需要加载 H5 网页，大概 1-2 秒。可以：
- 优化 H5 首屏加载速度
- 加个好看的启动页（当前已有 loading 动画）
- 使用 CDN 加速静态资源

## 更新 H5 地址

如果以后换了 H5 域名，只需要：

1. 修改 `app.js` 里的 `webViewUrl`
2. 小程序后台「业务域名」里添加新域名
3. 重新上传代码 → 提交审核 → 发布
