# 行业选股与多股对比子场景

本文件为 `stock-analysis-skill` 的子场景参考文件，定义行业选股和多股对比的完整执行流程。
当用户要求分析某个行业、推荐股票、对比多只股票时，AI 应加载本文件并严格遵循。

---

## 五、第二步：景气行业筛选标准

行业必须满足**至少一类核心景气逻辑**，才可进入个股分析；不满足则直接判定为"暂不关注"。

### 5.1 顺周期行业

受益于经济复苏、终端需求持续回升，量价同步修复，企业盈利进入上行通道。

**规则：** 必要条件（Hard Filter）

- PMI制造业连续 **>= 3个月** 处于荣枯线（50）以上
- 行业营收同比增速连续 **>= 2个季度** 改善
- 行业产能利用率 >= 75%（或环比持续提升）

---

### 5.2 行业景气度上行

供需格局改善，行业销量/产量持续提升，产能利用率紧张，行业整体盈利中枢抬升。

**规则：** 必要条件（Hard Filter）

- 行业销量/产量同比增速连续 **>= 2个季度** 为正且环比改善
- 行业整体毛利率连续 **>= 2个季度** 环比不降
- 头部企业（行业TOP3）产能利用率 >= 80%

---

### 5.3 产品提价周期

产品现货/期货价格持续上涨，传导顺畅，企业毛利率显著改善，业绩弹性明确。

**规则：** 必要条件（Hard Filter）

- 产品价格同比涨幅 **>= 10%** 且持续 **>= 1个季度**
- 价格涨幅 > 成本涨幅（价差扩大），即毛利率同比改善
- 下游需求未出现明显萎缩（下游行业PMI或开工率 >= 50）

---

### 5.4 困境反转行业

行业过去长期低迷，当前出现明确拐点信号。须至少满足**任意两项**拐点信号：

| 拐点类型 | 量化观察指标 |
|---------|------------|
| 政策端拐点 | 监管政策明确转向、重大产业扶持政策出台（以国务院/部委级别文件为准） |
| 需求端拐点 | 下游需求同比增速由负转正，且连续 >= 2个月 |
| 成本端拐点 | 主要原材料价格同比跌幅 >= 15%，且企业毛利率连续 >= 2个季度改善 |
| 供给端拐点 | 行业CR5提升 >= 5个百分点（同比），或行业企业数量同比减少 >= 10% |

## 六、第三步：个股三维深度分析

在优质行业内，从三个维度严格筛选个股。

### 6.1 盈利质量

- 营收、净利润增速趋势是否持续改善（连续 >= 2个季度环比/同比向好）
- 毛利率、净利率是否企稳或提升
- 经营性现金流是否健康，与利润匹配度（经营现金流/净利润 >= 0.7 为健康）
- 业绩是否具备持续性，排除一次性收益（非经常性损益占比 < 30%）

### 6.2 机构抱团程度

**正向信号**：机构持仓比例与变化趋势、近一季度机构增仓/减仓方向、券商研报覆盖密度与一致性预期、是否为板块核心龙头标的。

> ⚠️ **抱团拥挤度预警**
> 机构持股占流通盘比例 **>= 30%** 时触发拥挤预警，需额外评估：机构持仓集中度是否过高、是否存在"抱团瓦解"风险（如业绩不及预期引发集体减仓）、历史上该标的机构持仓峰值与回撤幅度。当抱团拥挤度触发预警时，**自动降低一档操作建议**（如从"重点关注"降为"跟踪观察"）。

### 6.3 筹码结构

- 筹码集中度与分散程度（股东户数变化趋势、前十大股东持股比例）
- 主力资金近期动向（大单净流入/流出、融资买入占比）
- 股价所处位置（低位筑底 / 中位整理 / 高位突破）
- 上方套牢盘压力、下方支撑强度（关键均线、筹码密集区）

## 十三、同行业全量标的深度比选

**[P0]** 必须对同一行业、同一主营业务的**所有可比上市公司**进行内部全量分析，从中选出风险收益比最优的 **5 只**，与用户询问的标的并列对比。

### 13.1 全量分析范围（内部执行，不输出）

> ⚠️ **内部执行 — 仅最终 Top 5 对比表呈现给用户**
> AI 必须搜索并分析同行业所有可比上市公司（通常 8–20 只），逐一完成以下分析，但逐只分析过程不向用户展示。仅将最终筛选出的 Top 5 对比结果呈现给用户。

**规则：** 每只标的内部分析清单

1. **基本面速览**：近两季度营收增速、净利润增速、毛利率、ROE、经营现金流/净利润
2. **估值定位**：PE(TTM)、PE 历史分位、PB、PEG
3. **机构动向**：机构持股比例、近一季度增/减仓方向、券商覆盖数
4. **赛道地位**：技术实力排名、市占率、护城河
5. **风险收益比**：按 8.3 方法估算每种情景概率，计算概率加权期望收益
6. **核心催化**：未来 2–3 个月关键催化事件

### 13.2 Top 5 筛选规则

从全量分析结果中按以下优先级排序，选出前 5 名：

1. **风险收益比**（权重 40%）：概率加权期望收益最高者优先
2. **盈利确定性**（权重 25%）：盈利可预测性、订单可见度
3. **估值性价比**（权重 20%）：PE 分位越低越好，PEG < 1 加分
4. **拥挤度安全边际**（权重 15%）：机构持仓 < 30% 优先，避免拥挤踩踏

### 13.3 用户可见对比输出格式

仅输出以下 Top 5 + 用户询问标的共计 6 行的对比表：

| 排名 | 股票 | PE(TTM) | PE分位 | Q1净利增速 | 机构持股 | 期望收益 | 盈亏比 |
|------|------|---------|--------|-----------|---------|---------|-------|
| *(实际分析中为具体数据)* | | | | | | | |

**规则：** 对比表必须包含的列

- **排名**：按风险收益比综合排序
- **股票名称 + 代码**
- **PE(TTM)**：当前估值
- **PE 历史分位**：估值在历史中的位置
- **近一季度净利润增速**：盈利弹性
- **机构持股比例**：抱团度
- **概率加权期望收益**：Σ(概率 x 涨跌幅)，来自 8.4 方法
- **盈亏比**：向上空间 / 向下风险

### 13.4 一句话替代推荐

在对比表后，必须给出明确推荐：

**规则：** 推荐格式

> "在 [行业] 赛道中，[XX 股票] 的风险收益比最优（期望收益 X%，盈亏比 X），相比您询问的 [原股票] 在 [具体维度] 上更优。若关注该赛道，建议优先考虑 [XX 股票]。"

若用户询问的标的已是 Top 1，则输出："您询问的 [XX] 在本赛道中风险收益比排名第 1，没有更优替代标的。"
